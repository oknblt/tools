File Tools

Collect · Merge · Edit · Compress PDFs — runs entirely in your browser

Collect PDF & Word files

Pick a main folder. Every PDF, DOC and DOCX inside its subfolders will be copied into a new Combined folder created in that location.

Your browser will ask permission to write files inside the folder you choose. Requires Chrome, Edge, or Opera.

Merge multiple PDFs into one

Pick the PDFs you want to combine — they'll be merged in the order shown.

Tip: select the PDFs from the Combined folder you generated in the Collect tab.

Edit a PDF

Rotate, delete, reorder pages. Add text or signatures. Redact sensitive areas.

Tap on the page to place
Drag to position, then apply.

Compress a PDF

Reduce PDF size by re-encoding embedded images at a lower resolution and quality. Text stays sharp; large photos get smaller.

Compression works by rasterizing each page at a target DPI and re-saving as JPEG inside a new PDF. Vector text is preserved as a flat image, which trades searchability for file size. Use the "High quality" preset if you need to keep text crisp at print resolution.
